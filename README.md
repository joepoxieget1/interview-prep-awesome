# interview-prep-awesome

## Description
A micro tool designed to help users prepare for interviews by providing resources and tips.

## Features
- Collection of common interview questions
- Tips for effective interviewing
- Interactive quizzes

## Technology Stack
- HTML
- CSS
- JavaScript

## Usage
1. Clone the repository
2. Open `index.html` in a web browser.
